# Sistema Híbrido de Transmissão Profissional

## Visão Geral
Sistema híbrido inspirado no vMix Call com integração OBS Studio, permitindo entradas via celular/browser, multicasting profissional e interface dark broadcast moderna.

## Áreas Principais

### 🎥 PROGRAM OUTPUT
Saída principal:
- YouTube
- Twitch
- TV
- RTMP
Formato: 16:9

### 📱 INSTAGRAM TX
Área dedicada:
- vertical
- TikTok/Reels/Instagram
Formato: 9:16

### 🎞 PREVIEW BUS
Miniaturas de:
- convidados
- celulares
- compartilhamento de tela
- mídia
- VT
- câmeras

### 🎚 AUDIO MIXER
Com:
- antirruído
- anti eco
- compressor
- limiter
- ducking
- noise gate
- VU meter realtime

### 🌐 CALL MANAGEMENT
Cada participante:
- entra via link
- celular ou navegador
- diretor aceita
- corta ao vivo
- muta
- coloca fullscreen

## Funcionalidades Pro
